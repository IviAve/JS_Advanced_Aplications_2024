requester.js праща request към сървъра, той не съхранява параметри, а само изпълнява различни функции.
userService.js праща данни за user-a към requester.js и я повиква с дадената функция, той съхранява всички параметри/данни за потребителя.
dataService.js праща данни за dashboard-а към requester.js и я повиква с дадената функция, тя съхранява всички параметри/данни в dashboard-а.
